# Decision Tree Classifier

A binary decision tree classifier built from scratch in Java. The tree is trained on labeled `TextBlock` data, where we recursively split on the feature with the greatest difference between samples.

## Features
- Train from labeled data or load a previously saved tree
- Serialize and deserialize trees in preorder format
- Accuracy reporting per label and overall

## Concepts demonstrated
- Binary tree construction and traversal
- Recursive algorithms
- File I/O and serialization
- Text-based classification without external ML libraries

## Requirements
Java 8+
